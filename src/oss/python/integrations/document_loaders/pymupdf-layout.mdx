## Setup

To access PyMuPDF Layout document loader you'll need to install the `langchain-pymupdf-layout` integration package.

### Credentials

No credentials are required to use PyMuPDFLayoutLoader.

If you want to get automated best in-class tracing of your model calls you can also set your [LangSmith](https://docs.smith.langchain.com/) API key by uncommenting below:

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### Installation

Install **langchain-pymupdf-layout**.

```python
pip install langchain-pymupdf-layout
```

## Initialization

Now we can instantiate our model object and load documents:

```python
from langchain_pymupdf_layout import PyMuPDFLayoutLoader

file_path = "https://www.adobe.com/support/products/enterprise/knowledgecenter/media/c4611_sample_explain.pdf"
loader = PyMuPDFLayoutLoader(file_path)
```

## Load

```python
docs = loader.load()
import pprint
pprint.pp(docs[0].metadata)
```
